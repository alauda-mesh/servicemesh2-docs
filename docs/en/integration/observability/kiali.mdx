# Alauda Build of Kiali

## Using Alauda Build of Kiali

Once you have added your application to the mesh, you can use Alauda Build of Kiali to view the data flow through your application.

### About Kiali

You can use Alauda Build of Kiali to view configurations, monitor traffic, and analyze traces in a single console. It is based on the open source [Kiali](https://github.com/kiali/kiali) project.

Alauda Build of Kiali is the management console for Alauda Service Mesh. It provides dashboards, observability, and robust configuration and validation capabilities. It shows the structure of your service mesh by inferring traffic topology and displays the health of your mesh. Kiali provides detailed metrics, powerful validation, access to Grafana, and strong integration with the distributed tracing components such as Jaeger or OpenTelemetry.
